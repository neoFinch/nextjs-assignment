# Pokemon App - NextJS Assessment

A Next.js 15 application that displays Pokemon data by fetching from the PokeAPI.

## Getting Started

### 1. Install dependencies

pnpm install

### 2. Set up environment variables

Create a `.env.local` file by copying from `.env.example`:


### 3. Run the development server

pnpm devOpen [http://localhost:3000](http://localhost:3000) in your browser.

## Testing

pnpm test
pnpm test:coverage## Tech Stack

- Next.js 15.5.6
- React 19
- TypeScript
- Tailwind CSS
- Jest + React Testing Library

## API

Uses [PokeAPI](https://pokeapi.co/) for Pokemon data. All API calls include error handling with user-friendly error messages.
